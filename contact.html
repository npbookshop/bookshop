1. Go to the web page you want to create a note for.
2. Select the "Make a Web Note" icon in the upper right corner of the Edge browser.
3. Select the "Pen" icon to draw on the page with a mouse, stylus, or your finger.
4. Use "Options" to change color and size. Select "Highlighter" to highlight text. Choose "Typed Note" tool to add text.
